# The Gap Scan

A leadership self-scan. For each of 18 leadership paradigms it puts two things on one line: what a recent, real situation called for, as you read it, and where your own instincts default. The space between them is the finding. There is no score and no archetype; either pole can be the right fit for a given situation, and "close" is the good result.

The questions are drawn from a book I am writing.

**Take it here: <https://chrisgagne.github.io/gap-scan/>** — or download [index.html](index.html) and open it locally, offline if you like.

## No data is collected or stored

The whole app is one self-contained HTML file. It makes no network requests, sets no cookies, and uses no storage of any kind; your answers live only in the open tab and are gone when you close it. You can verify this yourself: view source (no fetch, no beacon, no analytics, no external URLs), or run the file offline.

## Provenance

I built the original as part of my consulting practice. This version is stripped of branding and calls to action, published for the leadership communities I'm part of.
